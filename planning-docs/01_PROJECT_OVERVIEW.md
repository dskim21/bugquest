# 프로젝트 개요

## 서비스명
BugQuest

## 한 줄 소개
초보 개발자가 반복하는 에러와 실수를 기록하고, AI 설명과 퀘스트를 통해 성장 루프를 만드는 웹앱.

## 문제 정의
초보 개발자는 같은 에러를 반복해서 만나지만, 해결 과정을 체계적으로 기록하지 못한다.

## 해결 방법
에러를 기록하면 원인, 해결 방법, 실수 유형을 정리하고, 해당 실수를 줄이기 위한 작은 학습 퀘스트를 제공한다.

## 핵심 사용자
- 코딩 입문자
- 부트캠프 수강생
- React 초보 개발자
- 포트폴리오를 준비하는 주니어 개발자

## 차별점
일반 TODO 앱이 아니라, 개발 실수를 기반으로 개인 맞춤 학습 퀘스트를 생성한다.
