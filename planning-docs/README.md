# BugQuest

초보 개발자의 반복 실수를 기록하고, AI 설명과 성장 퀘스트로 연결하는 개발 성장 웹앱입니다.

## 핵심 컨셉

실수 → 원인 이해 → 복습 퀘스트 → 성장 기록

## 주요 기능

- 개발 에러/실수 기록
- 실수 유형 태그 분류
- AI 초보자 설명
- 오늘의 개발 퀘스트 추천
- 실수 통계 대시보드
- 레벨/XP 시스템

## 기술 스택

- React
- Vite
- Tailwind CSS
- React Router
- Zustand
- LocalStorage
- Recharts
- OpenAI API 선택 적용

## MVP 목표

로그인 없이 LocalStorage 기반으로 1~2주 안에 완성한다.
