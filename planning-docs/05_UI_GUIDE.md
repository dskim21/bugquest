# UI 가이드

## 분위기
- 개발자 성장 게임
- 다크모드 기반
- 카드형 UI
- 너무 귀엽기보다는 깔끔하고 몰입감 있게

## 메인 컬러
- 배경: dark navy / charcoal
- 포인트: violet 또는 cyan
- 상태 색상:
  - easy: green
  - normal: yellow
  - hard: red

## 핵심 컴포넌트
- Sidebar
- Header
- StatCard
- ErrorLogCard
- TagBadge
- QuestCard
- LevelProgress
- EmptyState
- SearchBar
- FilterTabs
