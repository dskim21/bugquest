# BugQuest

초보 개발자가 반복하는 에러와 실수를 기록하고, 복습 퀘스트를 통해 성장 루프를 만드는 웹앱입니다.

## 주요 기능

- 에러 로그 CRUD
- 태그/난이도 기반 검색 및 필터
- LocalStorage 데이터 저장
- Mock AI 에러 설명
- 실수 태그 기반 퀘스트 추천
- XP / Level 시스템
- 통계 차트
- 반응형 Sidebar UI

## 기술 스택

- React
- Vite
- Tailwind CSS
- React Router
- Zustand
- Recharts
- lucide-react
- LocalStorage

## MVP 특징

- 백엔드 없이 무료로 동작
- API 비용 없이 Mock AI 로직 사용
- Vercel 무료 배포 가능
- 1~2주 안에 완성 가능한 포트폴리오 범위
